Street View Publish Client

Antes de começar...

Use o botão abaixo para se autenticar com sua conta Google.

Este aplicativo permite enviar fotos 360° para o Street View e criar conexões entre elas.
Suas fotos precisam ter os metadados descritos em Metadados XMP de Photo Spheres para publicação. Além disso deve estar no formato JPEG, possuir no mínimo 7.5MP (3840 x 1920 px), ter no máximo 75MB e estar na proporção 2:1.
Uma boa sugestão é usar uma versão antiga do APK de câmera do Google pois este gera uma imagem compatível.
O projeto foi inspirado em PhotoSphereStudio
